# Queue Practice Problems – Python (OOP & DSA)

## Use Cases (UC)

- UC1 – Create a Queue class using list to perform basic enqueue and dequeue operations.
- UC2 – Implement peek (front) operation to view the first element of the queue.
- UC3 – Check if the queue is empty and handle underflow conditions.
- UC4 – Implement size method to return the number of elements in the queue.
- UC5 – Implement queue using collections.deque for optimized performance.
- UC6 – Reverse a queue using stack.
- UC7 – Generate binary numbers from 1 to N using queue.
- UC8 – Implement circular queue with fixed size.
- UC9 – Solve sliding window maximum problem using deque.
- UC10 – Implement queue using two stacks (advanced problem).
